# Optimal_routes_on_graphs_for_groups
# задача
начальные данные:
  1) координаты угроз
  2) точки входа и выхода для группы
  3) вероятности поражения от угрозы

цели:
  1)найти оптимальную численность группы
  2)маршрут группы
  3)чиссленность подгрупп в каждой точке маршрута

# пайплайн

  4) в каждой промежуточной точке $A_i$ есть вероятность $P_I$ ЧТО БПЛА пролетит дальше, необходимо найти оптимальный маршрут сходя из ограничений - в каждой промежуточной точке численность подгруппы больше 1, потери должны быть минимальны
например если вероятность в каждой точке равна 0.5, то оптимальным будет маршрут 3, начальная группа 16, потери 15:
